# Global Solution 1 - SERS
O SpaceEnergy Monitor é um sistema inteligente de monitoramento desenvolvido para uma missão espacial experimental. O projeto tem como objetivo receber, analisar e exibir dados simulados relacionados às condições operacionais da missão, permitindo a identificação de situações críticas e auxiliando na tomada de decisões automatizadas.

A solução aplica conceitos de programação, algoritmos, energia, potência, energias renováveis e sustentabilidade, simulando o funcionamento de um sistema de monitoramento utilizado em missões espaciais modernas.

## 🎯 Objetivo

Desenvolver uma plataforma capaz de:

- Monitorar dados simulados da missão;
- Analisar condições operacionais dos módulos;
- Gerar alertas automáticos;
- Realizar tomada de decisão básica;
- Exibir informações de forma organizada;
- Aplicar conceitos de sustentabilidade energética.

---

## ⚙️ Funcionalidades

### Monitoramento de Dados

O sistema realiza a leitura simulada de:

- Temperatura dos módulos;
- Nível de energia disponível;
- Status da comunicação;
- Estado operacional dos módulos.

### Geração de Alertas

Alertas automáticos são emitidos quando:

- A temperatura está fora da faixa ideal;
- O nível de energia está crítico;
- A comunicação é perdida;
- Um módulo apresenta falha.

### Tomada de Decisão

O sistema sugere ações corretivas para cada situação crítica detectada.

### Sustentabilidade

São apresentadas recomendações relacionadas ao uso eficiente da energia disponível na missão.
